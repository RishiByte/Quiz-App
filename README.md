# 🎮 Quiz App (Interactive)

A dynamic and engaging quiz application built using **React, JavaScript, HTML, and CSS**, powered by the **Open Trivia DB API**. This project fetches real-time quiz questions from an online database and allows users to test their knowledge across multiple categories and difficulty levels.

---

## 🚀 Features

* 🧠 **Real-time Questions** from Open Trivia DB API
* 🎯 Multiple categories (Science, Sports, History, etc.)
* ⚡ Difficulty levels (Easy, Medium, Hard)
* ⏱️ Interactive quiz experience with instant feedback
* 📊 Score tracking at the end of the quiz
* 🔄 Restart quiz option
* 💻 Fully responsive design

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript, HTML, CSS
* **API:** Open Trivia DB (https://opentdb.com/)

---



## 🔌 API Usage

The app fetches quiz questions using the Open Trivia DB API.

Example API endpoint:

```
https://opentdb.com/api.php?amount=10&category=18&difficulty=easy&type=multiple
```

### Parameters:

* `amount` → Number of questions
* `category` → Quiz category
* `difficulty` → easy / medium / hard
* `type` → multiple / boolean

---

## 🎯 How It Works

1. User selects quiz options (category, difficulty, number of questions)
2. App sends request to Open Trivia DB API
3. Questions are displayed one by one
4. User selects answers
5. Score is calculated and displayed at the end
